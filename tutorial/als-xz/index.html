ALS-XZ Rule

1. What You Will Master

In this section, you will learn the ALS-XZ Rule. You will learn how to find two separate Almost Locked Sets that are connected by a "collision" on one specific digit (the Restricted Common Candidate). You will use this collision to prove that a different digit (z) must exist in one of the two sets, allowing you to eliminate z from any cell that "sees" both sets.

2. Main Content

The Components

To perform an ALS-XZ move, you need three ingredients:

  • ALS A: A group of cells acting as a set (e.g., {1, 2, x, z}).
  • ALS B: A second group of cells acting as a set (e.g., {3, 4, x, z}).
  • RCC (Restricted Common Candidate — 'x'): A digit that appears in both sets and links them.

Condition: The 'x' candidates in ALS A must see the 'x' candidates in ALS B. This creates a strong link between the sets (they cannot both contain 'x').

The Logic: The Seesaw

Because the sets see each other, we cannot have 'x' in Set A and 'x' in Set B simultaneously. Therefore, at least one of the sets does not contain 'x'.

  • Scenario 1: Set A does NOT contain 'x' → Set A becomes a locked set for its other digits (including z) → z is true in Set A.
  • Scenario 2: Set B does NOT contain 'x' → Set B becomes a locked set for its other digits (including z) → z is true in Set B.

Conclusion: In every case, the digit z must exist in Set A or Set B.

The Elimination (The 'z')

The digit z is the elimination target. It is the digit that appears in both sets (but is not the link).

Target: Look for any cell outside the sets.

Condition: If that cell sees all instances of 'z' in Set A and 'z' in Set B...

Action: Eliminate 'z' from that outside cell.

Example

ALS A: {1, 9} and {1, 2, 9} in Row 1 (candidates: 1, 2, 9).

ALS B: {2, 7} and {2, 7, 9} in Row 3 (candidates: 2, 7, 9).

Link (RCC): The digit 9, with the '9's in the two ALS seeing each other (e.g., in the same column). One set must lose 9.

Target (z): The digit 2 (present in both). Any cell that sees the '2's in ALS A and ALS B cannot be 2.

Doubly Linked ALS-XZ (Optional)

If the two sets share two RCCs (e.g., both have x and y, and both pairs see each other), this forms a naked pair relationship between the sets. You can eliminate candidates from the sets themselves (mutual elimination). Rare but deadly.

3. Section Summary

The ALS-XZ rule is a "bent naked pair" made of groups rather than single cells.

  • If Set A has the link, Set B forces the target.
  • If Set B has the link, Set A forces the target.
  • Therefore, the target is inescapable.

Once you can identify ALS groups (N cells with N+1 candidates), spotting the links becomes a fun matching game.
