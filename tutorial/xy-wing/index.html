XY-Wing / Y-Wing

1. What You Will Master

In this section, you will learn the XY-Wing (often called the Y-Wing). This is the most fundamental multi-digit technique. You will learn how to spot a "Bent Triple"—a chain of three cells that forces a specific number to appear at one end or the other, allowing you to eliminate that number from the intersection.

2. Main Content

The Components

An XY-Wing is built from exactly three Bi-Value Cells.

The Pivot (The Stem): The central cell. It shares a unit (Row, Col, or Box) with both wings. Let's call its candidates {X, Y}.

Wing 1: A cell seeing the Pivot. It contains {X, Z}.

Wing 2: Another cell seeing the Pivot. It contains {Y, Z}.

Key Requirement: The two Wings must contain a common candidate (the "Z"), but they do not need to see each other. They only need to see the Pivot.

The Logic: The "Fork"

Let's trace the logic starting from the Pivot {X, Y}:

Scenario A: If the Pivot is X, then Wing 1 creates a Z (because it was XZ).

Scenario B: If the Pivot is Y, then Wing 2 creates a Z (because it was YZ).

Conclusion: Since the Pivot must be either X or Y, one of the two Wings guarantees a Z. We don't know which one, but we know the 'Z' exists in one of those two endpoints.

The Elimination

Because "Z" must exist in Wing 1 OR Wing 2:

Identify the common candidate Z.

Find any cell on the grid that is seen by BOTH Wing 1 and Wing 2.

Action: Eliminate the candidate Z from that intersection cell.

Note: If that cell contained a Z, it would break both reality A and reality B, which is impossible.

Example

Pivot: R1C1 {1, 2}

Wing 1: R1C5 {1, 5} (Shares Row 1)

Wing 2: R3C1 {2, 5} (Shares Col 1)

Analysis: If Pivot is 1, Wing 1 is 5. If Pivot is 2, Wing 2 is 5.

Elimination: Any cell that sees both R1C5 and R3C1 cannot contain a 5. (In this case, cell R3C5 is the kill zone).

3. Section Summary

The XY-Wing is powerful because it connects disjointed parts of the grid. The Wings might be far apart, but the Pivot acts as a hinge connecting them. Whenever you are stuck on a hard puzzle, scan for Bi-Value Cells. If you find a cell with only two options, ask: "Is this a Pivot? Does it connect to other bi-value cells?"
