XYZ-Wing

1. What You Will Master

In this section, you will learn the XYZ-Wing. You will discover what happens when an XY-Wing Pivot has an extra candidate (becoming a Tri-Value cell). You will learn why this extra candidate restricts the "Kill Zone" significantly, making this technique more localized but still essential for breaking tough boxes.

2. Main Content

The Components

Unlike the XY-Wing (which uses 3 Bi-Value cells), the XYZ-Wing uses one Tri-Value cell and two Bi-Value cells.

The Pivot: The central cell. It contains {X, Y, Z}. (e.g., 1, 4, 9).

Wing 1: Sees the Pivot. Contains {X, Z}. (e.g., 1, 9).

Wing 2: Sees the Pivot. Contains {Y, Z}. (e.g., 4, 9).

Crucial Difference: In an XY-Wing, the Pivot is just {X, Y}. Here, the Pivot also contains the common candidate Z.

The Logic: The "Triple Threat"

Consider the common candidate Z (9 in our example). There are three possibilities for the Pivot:

Pivot is X (1): Then Wing 1 becomes Z (9).

Pivot is Y (4): Then Wing 2 becomes Z (9).

Pivot is Z (9): Then the Pivot itself is Z (9).

Conclusion: In every possible universe, a Z (9) exists in either Wing 1, Wing 2, or the Pivot itself.

The Elimination

This is where it gets tricky. Because Z might be in the Pivot itself, we cannot eliminate from anything that only sees the wings.

The Rule: You can only eliminate Z from a cell that sees all three components:

It must see Wing 1.

It must see Wing 2.

It must see the Pivot.

Result: Usually, the XYZ-Wing only eliminates candidates within the same Box as the Pivot (especially cells aligned with the external wing). The "Kill Zone" is much smaller compared to the XY-Wing.

How to Spot It

Find a Pivot with 3 numbers (e.g., 1, 4, 9).

Check if there is a neighbor with (1, 9) and another with (4, 9).

Check geometry: Does the Pivot share a Box with one Wing and a Row/Col with the other? This is the standard shape.

3. Section Summary

The XYZ-Wing is a "cluttered" XY-Wing. Because the Pivot holds Z, Z is not forced solely to the wings; it might stay in the center. Therefore, you can only eliminate candidates that are surrounded by the entire formation. It is a weaker technique than the XY-Wing, but it often appears in situations where the XY-Wing fails.
