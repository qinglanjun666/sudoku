Two-String Kite

1. What You Will Master

In this section, you will learn the Two-String Kite. You will learn how to use a specific Box as a "cornerstone" to connect a Row and a Column. This technique allows you to prove that a candidate must exist in one of two "tails" that hang out of the box, eliminating the candidate where those tails intersect.

2. Main Content

The Components

A Two-String Kite requires one digit (e.g., '3') and three parts:

The Box: A 3x3 Box that acts as the connector (e.g., Box 3).

String 1 (The Row): A Row passing through the box that has a Strong Link (exactly two '3's). One '3' is inside the box, the other '3' is outside.

String 2 (The Column): A Column passing through the box that also has a Strong Link. One '3' is inside the box, the other '3' is outside.

The Crucial Link: The two candidates inside the box must "see" each other (share a row or column) or be connected by the box's logic so that they cannot both be true.

The Logic: "The Chain Reaction"

Let's name the points:

A: The '3' in the Row, outside the box (Tail 1).

B: The '3' in the Row, inside the box.

C: The '3' in the Column, inside the box.

D: The '3' in the Column, outside the box (Tail 2).

The Chain:

If A is False, then B must be True (Strong Link).

If B is True, then C must be False (they see each other in the box).

If C is False, then D must be True (Strong Link).

Conclusion: If A is False, D is True. Therefore, it is impossible for A and D to both be False. One of them is the real digit.

The Elimination

Because the digit must be in Tail 1 (A) or Tail 2 (D):

Identify the two outside candidates (the ends of the strings).

Find the cell where Tail 1's Row and Tail 2's Column intersect.

Action: Eliminate the candidate from that intersection cell.

Visual Check

Look at a corner box (like Box 3, 7, or 9).

Does a Row enter it with a 2-candidate limit?

Does a Column enter it with a 2-candidate limit?

Do the ends inside the box connect?

If yes, look at the ends outside the box. Where do they meet? Shoot that spot.

3. Section Summary

The Two-String Kite is a classic "Turbot Fish" pattern (a chain of 3 strong links). It is useful because it feels natural: "A Row comes in, hits the Box, turns 90 degrees, and leaves as a Column." This change of direction (Row to Column) is what allows the intersection elimination.
