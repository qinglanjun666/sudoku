Continuous Loops / X-Cycles

1. What You Will Master

In this section, you will learn the Continuous Loop (often called an X-Cycle when dealing with a single digit). You will discover the stabilizing power of a closed circuit. Unlike a standard chain which acts like a "gun" shooting a target, a Continuous Loop acts like a "force field," upgrading every link in the chain and eliminating candidates surrounding it.

2. Main Content

The Concept

Recall the AIC from the previous section: Start = A - B = C - End. This proved that Start and End are linked.

A Continuous Loop occurs when the End is connected back to the Start with a link that maintains the alternating rhythm (Strong-Weak-Strong-Weak).

Structure: A = B - C = D - A

Condition: The number of nodes must be even.

The Logic: "The Perpetual Motion Machine"

Because the loop is closed and alternating:

  • If A is ON 鈫?B is OFF 鈫?C is ON 鈫?D is OFF 鈫?A is ON. (Consistent).
  • If A is OFF 鈫?B is ON 鈫?C is OFF 鈫?D is ON 鈫?A is OFF. (Consistent).

The Transformation

Because the logic holds up in both directions (clockwise and counter-clockwise), the distinction between "Strong" and "Weak" links inside the loop disappears.

Every Weak Link in the loop becomes a Strong Link.

Normally, A - B means "at most one is true." In a loop, it becomes "exactly one is true."

The Elimination

Since every link in the loop effectively becomes a Strong Link, we can eliminate candidates located outside the loop that interfere with the loop's connections.

Rule 1: The Weak Link Elimination
Look at any Weak Link connection inside the loop (e.g., the relationship between B and C above). Since B and C cannot both be False (they are now strongly linked by the loop), any outside candidate that sees BOTH B and C can be eliminated. Why? If the outsider were true, it would force both B and C to be False, breaking the loop.

Rule 2: The Locked Candidates
Sometimes, the loop proves that a specific candidate must be part of the solution in a specific row or column, allowing you to clear the rest of that house.

Example Scenario (X-Cycle)

You find a chain of the digit '5'. Row 1 (Strong) 鈫?Col 9 (Weak) 鈫?Row 8 (Strong) 鈫?Col 2 (Weak) 鈫?Back to Row 1.

Result: The '5's in Col 9 and Col 2 are now "Strongly Linked" pairs.

Action: Any other '5' that sees the pair in Col 9 or the pair in Col 2 can be removed.

How to Spot It

Continuous loops are often found by accident while looking for an AIC. Start building an AIC: Start = Node A - Node B... Suddenly, you realize your last node lands in a spot that connects strongly back to your Start. Stop! You have found a loop. Don't look for a target cell. Instead, look at the links of your loop. Eliminate anything "weak" looking at your links.

3. Section Summary

Continuous Loops are the most aesthetically pleasing patterns in Sudoku. They represent a perfectly balanced system. When you find one, the loop "crystallizes"鈥攅very candidate in the loop becomes essential, and the weak links between them become rigid bars of steel. Any external candidate trying to "break" a link by seeing both ends of it is instantly eliminated.
