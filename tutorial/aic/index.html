AIC (Alternating Inference Chains)

1. What You Will Master

In this section, you will learn the Alternating Inference Chain (AIC). This is the "Grand Unified Theory" of Sudoku. You will learn how to build chains that mix and match different types of links—connecting a Strong Link in a Row to a Bi-Value Cell to a Strong Link in a Box. You will master the universal notation of chains and the ability to find eliminations in unstructured chaos.

2. Main Content

The Concept

An AIC is a chain that alternates strictly between Strong Links and Weak Links.

Recall Section 13.1:

Strong Link (=): If A is False, B is True. (Cannot both be False).

Weak Link (-): If A is True, B is False. (Cannot both be True).

The Rules of the Chain

To build a valid AIC, you must follow this rhythm:

Node A = Node B - Node C = Node D - Node E ...

Start with a Candidate: Let Candidate A be assumed OFF (False).

Strong Link: Because A is OFF, B must be ON.

Weak Link: Because B is ON, C must be OFF.

Strong Link: Because C is OFF, D must be ON.

Conclusion: If we assume Start (A) is False, the End (D) is forced to be True. Therefore, A and D cannot both be False.

Mixing Node Types

The power of AIC is that Nodes can be anything:

Type 1: A specific candidate in a cell (e.g., the 7 in R1C1).

Type 2: A logic state (e.g., the digit 7 in Row 1).

You can jump between them:

Link 1: (1)R1C1 is strongly linked to (1)R1C5 (Row Link).

Link 2: (1)R1C5 is weakly linked to (4)R1C5 (Cell Link / Bi-Value).

Link 3: (4)R1C5 is strongly linked to (4)R9C5 (Column Link).

This allows you to weave through numbers and houses seamlessly.

The Elimination

The elimination target relies on the Strong Link property between the Start and End we just proved.

The Logic: Since Start is False → End is True (and vice versa by reverse logic), Start and End behave like a Strong Link.

The Kill: Any candidate that is weakly linked to both the Start and the End can be eliminated.

Translation: Find a candidate that sees the Start and sees the End. If that candidate were True, it would render both Start and End False. But Start and End cannot both be False. Therefore, the common neighbor is impossible.

Standard Notation

Sudoku communities use a shorthand. Learn to read it:

(1)R1C1 = (1)R1C5 - (4)R1C5 = (4)R9C5

= : Strong Link

- : Weak Link

(x)RxCx : Candidate x in Row r, Column c.

How to Spot It

Find a Strong Link: Always start here. "Row 5 has only two 8s."

Extend Weakly: From one end of the 8s, look for a weak connection. "This 8 sees a Bi-Value Cell {8,9}."

Extend Strongly: "If that cell isn't 8, it is 9."

Repeat: "Where does that 9 go?" Look for another strong link.

Close the Loop: Try to land somewhere that sees your starting point.

3. Section Summary

AIC is not a specific shape; it is a process. It is the ability to say: "If this is off, then that is on, which means this is off, which means that is on..." Every named pattern you learned (X-Wing, XY-Chain, Skyscraper) is just an AIC. X-Wing: (A) = (B) - (C) = (D). XY-Chain: (A) - (B) = (C) - (D) = ... Master AIC and you no longer need to memorize names—you read lines of force on the board.
