ALS (Almost Locked Sets)

1. What You Will Master

In this section, you will learn the definition of an Almost Locked Set (ALS). You will learn to recognize groups of cells that are "overloaded" by exactly one digit. This is not an elimination technique by itself; it is a building block. Just as a "Strong Link" is a building block for an X-Wing, an ALS is a building block for the most advanced eliminations in Sudoku (ALS-XZ, Sue de Coq).

2. Main Content

The Definition

To understand an Almost Locked Set, remember what a Locked Set is.

Locked Set (Naked Subset): N cells containing exactly N candidates.

Example: Two cells containing only {1, 2}. (2 cells, 2 digits). State: These are "full." No other digits can enter.

Almost Locked Set (ALS): N cells containing exactly N+1 candidates.

Example: Two cells containing {1, 2} and {1, 2, 3}. (2 cells, 3 digits). State: These are "one step away from being full." They have one degree of freedom.

The Logic: "The Loaded Spring"

Think of an ALS as a loaded spring. It is holding one extra number.

Action: If you can prove the "extra" digit is false for the set, the spring snaps shut.

Consequence: The remaining digits form a Locked Set (Naked Pair/Triple/Quad), triggering eliminations in the house.

Example Setup

Imagine three cells in the same Row:

  • Cell A: {1, 2}
  • Cell B: {2, 3}
  • Cell C: {1, 3, 9}

Analysis: Cells: 3 (A, B, C). Candidates: 4 ({1, 2, 3, 9}). Formula: N cells, N+1 candidates. This is an ALS.

Why useful? Treat the entire ALS as a single Node in a chain.

Logic statement: "Either the 9 is true, OR the set {1, 2, 3} becomes a Locked Naked Triple." This enables inferences: "If 9 is false, then 1, 2, and 3 are locked in these cells, so 1, 2, and 3 can be eliminated from the rest of the row."

Restricted Common Candidate (RCC)

This is crucial for the next section. When you have two ALS sets, look for a digit that overlaps between them.

If ALS #1 has {1, 2, 3} and ALS #2 has {4, 5, 3}, the digit 3 acts as the bridge. If 3 is excluded from ALS #1, it may be forced into ALS #2.

3. Section Summary

  • An ALS is a "virtual cell."
  • A bi-value cell has 1 cell and 2 candidates ("If not A, then B").
  • An ALS has N cells and N+1 candidates ("If not the extra digit, then the rest are locked").

You generally don't scan the board for random ALS patterns. You spot them when clusters of cells share similar digits. In the next section, we'll rub two ALS groups together to create fire.
