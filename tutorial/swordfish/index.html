Swordfish

1. What You Will Master

In this section, you will learn the Swordfish. This is a pattern involving three rows (or three columns) instead of two. You will learn to recognize when a specific digit is restricted across three parallel lines in a way that locks them together, allowing for massive eliminations across the grid.

2. Main Content

The Concept

A Swordfish occurs when a specific digit (e.g., '5') appears in exactly three Rows, and within those rows, the candidate falls into exactly three Columns.

Base Set: 3 Rows.

Cover Set: 3 Columns.

Note: Not every row needs to have all 3 columns filled. It can be a "partial" grid (e.g., Row A has cols 1&2, Row B has 2&3, Row C has 1&3). As long as the total unique columns involved is 3, it works.

The Logic: The 3×3 Grid

Just like the X-Wing (2×2), the Swordfish creates a closed loop. Because we have 3 Rows that need to place a '5', and they are confined to only 3 Columns, each Column will receive exactly one '5' from these rows.

Therefore, these 3 Rows consume the 3 Columns entirely regarding the number '5'. No other row can place a '5' in these specific columns.

The Elimination

Identify the Base: Find 3 Rows where the digit exists only in Columns A, B, and C.

Identify the Target: Look at the full length of Columns A, B, and C.

Action: Eliminate the candidate '5' from any cell in Columns A, B, and C that is not part of your original 3 Rows.

Vertical Swordfish

Like the X-Wing, this works in reverse.

If you find 3 Columns where the digit is restricted to the same 3 Rows...

...Then you eliminate that digit from the rest of those Rows.

How to Spot It

This is hard to see with the naked eye.

Filter: Highlight the number you are checking.

Count: Look for rows that have only 2 or 3 candidates of that number.

Match: Can you find three such rows that align perfectly? (e.g., Row 1, 4, 8 all line up on Cols 2, 5, 9).

3. Section Summary

The Swordfish is a "Triple X-Wing." It is a pattern where 3 lines team up to claim 3 perpendicular lines. The rule is simple: N Rows claimed by N Columns. If 3 Rows limit a number to 3 Columns, those columns are "dead" to outsiders. Mastering the Swordfish is a major step in seeing the grid as a matrix rather than just a collection of boxes.
